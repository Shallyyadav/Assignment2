Introduction of Html

HTML stands for Hyper Text Markup Language. It is used for creating web pages. It describes the structure of a web page.

Html Versions

HTML 1.0 was the first version of HTML,1993. HTML 2.0 was released in 1995. Then HTML 3.0 released in 1996. HTML 4.0 released in 1997. HTML 4.0.1 in 1999. XHTML was introduced in 2000. HTML5 is the latest version of HTML, and it was released in 2014.
